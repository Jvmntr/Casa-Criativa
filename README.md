### Para rodar a aplicação:

<ul>
<li> Clone esse repositório. </li>
<li> Execute npm install dentro da pasta criada. </li>
<li> Rode npm run dev. </li>
<li> Acesse a aplicação, localhost:3000. </li>
</ul>
